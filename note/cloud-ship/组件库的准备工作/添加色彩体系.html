# 颜色体系

系统色板 - 基础色板 + 中性色版 需要有 主题颜色+辅助颜色（又称功能色）

# 颜色结构挑选

比如选定好 蓝色、绿色、黄色、红色、深灰色 代表着primary success warning danger info

# 安装依赖

npm insatll node-sass --save

# 添加全局css变量

创建 src/styles/_variables.scss

使用default关键字后，这些变量拥有了默认值，当用户重新定义之后,那么变量则不使用默认值。

$white: #fff !default;
$gray-100: #f8f9fa !default;
$gray-200: #e9ecef !default;
$gray-300: #dee2e6 !default;
$gray-400: #ced4da !default;
$gray-500: #adb5db !default;
$gray-600: #6c757d !default;
$gray-700: #495057 !default;
$gray-800: #343a40 !default;
$gray-900: #212529 !default;
$black: #000 !default;
